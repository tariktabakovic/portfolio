BeatsByMe

BeatsByMe is a front-end project I worked on during my cohert with two other students. We all shared similar passions for music, so we wanted to build our project around the creation of music. We did so by using the following resources: HTML, CSS, JavaScript, p5.js(API), and freesound.org(API). We incorperated our sounds by uploading them to freesound.org, and then fetched them into our app. We also installed the p5 sound library which allowed us to load and play sounds in specific manners. The p5 library also gave us the ability to incorperate a 16-step audio sequencer. Through this sequencer the user is able to create their own unique sound patterns, and run it.

WhoIsGrandmaster?

WhoIsGrandmaster is a react based project that allows users to play a game of chess. I've decided to build this app because I've been fairly consumed by the game chess as of recently. I also thought that building it with React would be relativly easier than building it in any other framework. In addition to allowing users the option of playing, there is also game information displayed next to it. Another requirement for this project was to incorperate data from an API. In my case I used the API from chess.com, and fetched the top 5 ranked players from the sites' Blitz category. I enjoyed building this app, and was pleased with the simple yet effective layout of it.
